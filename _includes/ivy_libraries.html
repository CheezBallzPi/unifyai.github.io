Ivy Libraries

Mechanics

Orientation, position, and pose conversions. Co-ordinate transformations. Some silly text.

Vision

Voxel grids, rendering, optical flow, triangulation, SDFs and more.

Robot

Gradient-based motion planning for mobile robots and manipulators.

Gym

Differentiable RL gym environments, solvable via trajectory optimization.

Memory

Neural memory modules, such as neural turing machines and more.

Builder

Tools for building networks, trainers and data loaders in a few lines of code.

Models

Pre-trained models + implementations of well-known networks.

Ecosystem

Collection of third-Part libraries using Ivy to support all frameworks.
