Any User

Example

You have written a PyTorch library, and released this on GitHub

JAX, TensorFlow, MXNet and other ML developers cannot use your library

Your library is only useful to a subset of all ML developers

Current Solution

Limit your audience to users of your chosen framework, in this case PyTorch

Otherwise, create a fork and duplicate all of your work, re-implementing the entire library in the new framework

Ivy Solution

Open up your library to all ML developers simultaneously, without any code changes

Show how Ivy function decorators can be used to transpile any function in your library into any framework
